# 🌤 Sri Lanka Weather App

![App Screenshot](screenshot.png)

A desktop application that shows real‑time weather for **all 9 provinces of Sri Lanka**. Built with Python, Tkinter, and the OpenWeatherMap API.

## Features

- 50+ cities from every province
- Simple dropdown selection (no typing)
- Shows: temperature, feels like, humidity, wind speed, description
- Modern dark theme

## How to Run (from source)

1. **Get a free API key** from [OpenWeatherMap](https://home.openweathermap.org/api_keys)

2. **Clone this repository**  
   `git clone https://github.com/sudheera-hash/SriLankaWeatherApp.git`

3. **Create a `.env` file** in the project folder and add your key:  
   `WEATHER_API_KEY=your_api_key_here`  
   (Replace `your_api_key_here` with the key you received)

4. **Install dependencies**  
   `pip install -r requirements.txt`

5. **Run the app**  
   `python app.py`

## Technologies

Python 3, Tkinter, OpenWeatherMap API, python‑dotenv
